# cex.io-price-history

Dev environment

## Install Dependencies (for both client & server)

```bash
npm install
```

## Run Dev Server

```bash
npm run dev
```

## Run Dev Client (navigate to the Client folder)

```bash
npm run start
```

## Open the cex.js config file and enter your own CEX.IO ClientId, ApiKey and ApiSecret

module.exports = {
ClientId: "",
ApiKey: "",
ApiSecret: ""
};
